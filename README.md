# Superstore Sales Analysis

## Project Overview
This project analyzes retail sales data to identify profitable products and sales trends across different regions.

## Tools Used
- Excel
- Power BI
- Data Analysis

## Dataset
Superstore sales dataset containing product, category and regional sales information.

## Key Insights
- Identified profitable product categories
- Analyzed regional sales performance
- Detected loss-making product segments
